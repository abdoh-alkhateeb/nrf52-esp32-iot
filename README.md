# nRF52 + ESP32 IoT System

This project demonstrates a simple IoT system using an nRF52 microcontroller as a BLE sensor node and an ESP32 as a BLE-to-Wi-Fi gateway that also hosts a local web server. Sensor data is transmitted from the nRF52 via Bluetooth Low Energy (BLE) to the ESP32, which serves the data on a web interface accessible from any device on the same network for real-time monitoring.
